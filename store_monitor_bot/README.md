## 🚀 طريقة التشغيل

### الطريقة الأولى: تشغيل مباشر

```bash
# 1. استنسخ المشروع
git clone <your-repo-url>
cd store_monitor_bot

# 2. ثبّت المتطلبات
pip install -r requirements.txt
playwright install chromium

# 3. انسخ ملف الإعدادات
cp .env.example .env

# 4. عدّل .env وأضف مفاتيحك (انظر أدناه)
nano .env

# 5. شغّل قاعدة البيانات (أو استخدم Docker)
# PostgreSQL يجب أن يكون شغّالاً على localhost:5432

# 6. شغّل البوت
python main.py
```

### الطريقة الثانية: Docker (الأسهل)

```bash
# 1. انسخ ملف الإعدادات وعدّله
cp .env.example .env
# عدّل .env وأضف TELEGRAM_BOT_TOKEN على الأقل

# 2. شغّل كل شيء بأمر واحد
docker-compose up -d

# 3. شاهد اللوغ
docker-compose logs -f bot
```

## 🔑 إضافة مفاتيح API

### 1. مفتاح بوت تيليغرام (إلزامي)

1. افتح تيليغرام وابحث عن `@BotFather`
2. أرسل `/newbot`
3. اتبع التعليمات واحصل على التوكن
4. ضعه في `.env`:
   ```
   TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHI...
   ```

### 2. معرف مجموعة الإدارة (إلزامي)

1. اصنع مجموعة خاصة في تيليغرام
2. أضف البوت للمجموعة وجعله مشرفاً
3. ابعث أي رسالة في المجموعة
4. افتح: `https://api.telegram.org/bot<TOKEN>/getUpdates`
5. ابحث عن `"chat":{"id":` واحفظ الرقم (سيكون سالباً)
6. ضعه في `.env`:
   ```
   ADMIN_GROUP_ID=-1001234567890
   ```

### 3. معرف المديرين

1. افتح `@userinfobot` في تيليغرام
2. ابعث `/start` واحصل على User ID
3. ضعه في `.env`:
   ```
   ADMIN_USER_IDS=123456789
   ```

### 4. قاعدة البيانات

```
DATABASE_URL=postgresql://postgres:كلمة_المرور@localhost:5432/store_monitor
```

## 📖 شرح الملفات

### `config/settings.py`
كل إعدادات المشروع. **الملف الوحيد الذي تحتاج لتعديله** إضافة للـ `.env`.
- حدود الخطط (عدد المنتجات، سرعة الفحص)
- إعدادات تقييم الفرص
- إعدادات مكافحة الحظر

### `db/models.py`
تعريف جداول قاعدة البيانات:
- `User` - المستخدمون
- `Product` - المنتجات
- `UserProduct` - ربط المستخدم بالمنتجات
- `PriceHistory` - سجل الأسعار
- `Opportunity` - الفرص المكتشفة
- وغيرها...

### `core/monitor.py`
القلب الحقيقي للنظام:
- يفحص المنتجات دورياً
- يكتشف التغيرات
- يحسب نقاط الفرص (0-100)
- يرسل الفرص للإدارة

### `core/connectors/`
موصلات المتاجر:
- `amazon.py` - يدعم amazon.com, amazon.sa, amazon.ae وغيرها
- `generic.py` - يدعم Shopify, WooCommerce, وأي موقع آخر

### `admin/dashboard.py`
لوحة الإدارة الويب على `http://localhost:8000`
- إحصائيات في الوقت الحقيقي
- إدارة الفرص
- عرض المستخدمين

## 💡 نصائح للاستخدام

1. **ابدأ بأمازون فقط** - الأكثر استقراراً وأسهل دعماً
